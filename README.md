# Universal PDF & Image Compressor

A Python tool to compress PDFs using **Ghostscript** and images using **Pillow**. 

## Features
- **Auto-detection:** Automatically switches between Ghostscript (PDF) and Pillow (Images).
- **Interactive Menu:** Choose between 4 levels of compression.
- **Path Cleaning:** Handles Windows paths with quotes automatically.

## Requirements
1. **Ghostscript:** Must be installed (update the `GS_PATH` in the script).
2. **Pillow:** `pip install pillow`

## How to use
Run `python "PDF and Image Compressor.py"` and follow the on-screen prompts!
